# Code of Conduct

## Our Goal
This project is dedicated to creating an educational game about the RISC-V architecture. Our primary objective is to maintain a constructive, professional, and drama-free environment focused entirely on hardware simulation, game design, and software engineering.

## Expected Behavior
* **Focus on Technical Value:** Keep discussions relevant to the repository (code, architecture, hardware, bugs, and game mechanics).
* **Be Professional:** Critique ideas, logic, and implementation - not the people behind them. 
* **Constructive Feedback:** Provide objective, reasoned, and clear feedback during code reviews and discussions.

## Unacceptable Behavior
* **Harassment and Personal Attacks:** Insults, threats, personal attacks, or aggressive behavior toward any contributor.
* **Off-Topic & Political Debates:** Bringing political, ideological, or social debates into issue trackers, pull requests, or project discussions.
* **Trolling and Spam:** Intentionally derailing technical conversations, spamming, or opening bad-faith issues.

## Scope & Enforcement
This Code of Conduct applies to all project spaces, including GitHub Issues, Pull Requests, and Discussions.

Project maintainers have the right and responsibility to remove comments, close issues, or ban users who disrupt the productive engineering environment of this project.
